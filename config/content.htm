關於git與git指令

git的基本指令

cd ---------------------------------切換目錄，後面加 / 可以進入非同資料夾目錄中的資料夾，加 .. 可以回到上一層

git clone 網址  -----------------這個指令可以將指令網址中的資料，下載至近端

git add .  -------------------------將全部的改動做一次的快照(暫存)

git commit -m "改版名"   ---建立一個名為"改版名"的改版(將add檔案打包)

git push  -------------------------將改版上傳至遠端

git checkout 分支名or改版號   ---前往分支or改版

git pull  ---------------------------將近端網站和遠端網站進行比對，新增近端沒有的檔案

git merge 分支名------------將"分支名"中的檔案合併到當前分支
